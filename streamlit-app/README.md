# Document Processing Streamlit App

Simple Streamlit frontend for the Document Processing API.

## Setup

```bash
cd streamlit-app
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

The app will open in your browser at http://localhost:8501

## Features

- ✅ User login/register
- ✅ Document upload
- ✅ Real-time processing with AI agents
- ✅ View extraction results
- ✅ Processing history
- ✅ Clean, simple interface

## Requirements

- Backend must be running (with ngrok)
- Update `.env` with your ngrok URL
